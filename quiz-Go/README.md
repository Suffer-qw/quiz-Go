# Quiz-Go

Минимальное веб‑приложение для прохождения тестов.

## Стек

- фронт: HTML + JS (без фреймворков)
- бэк: Go (`net/http`)
- БД: SQLite

## Запуск

```bash
go mod tidy
go run ./cmd/server
```

Откройте `http://localhost:8000`.

## Учётные записи (первый запуск)

- **admin** / `admin123`
- **user** / `user123`

## Возможности

- регистрация/вход/выход
- роли `user` и `admin`
- каталог тестов
- прохождение теста (4 варианта ответа на вопрос)
- история: 10 последних попыток со статистикой
- админ: создание нового теста через JSON
